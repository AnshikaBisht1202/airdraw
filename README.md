# AirDraw

AirDraw is a computer vision application that enables users to draw in the air using hand gestures captured through a webcam. Built with MediaPipe and OpenCV, it tracks the user's index finger in real time and converts its movement into digital strokes on a virtual canvas.

## Features
- Real-time hand tracking
- Air-based drawing
- Virtual canvas
- Touchless interaction
- Smooth gesture recognition

## 🛠️ Technologies
- Python
- OpenCV
- MediaPipe
- NumPy
